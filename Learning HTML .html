What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the World Wide Web.
It uses tags and elements to tell a web browser how to display text, images, links, and videos.
HTML is the skeleton of a webpage.

What Does HTML Do?

HTML organizes content using tags.
It tells the browser what is a heading, paragraph, image, link, table, or form.

Why is HTML Important?

HTML is important because:
• All websites use it
• It is the base of web development
• Easy to learn for beginners
• Works on all browsers and devices

Basic HTML Structure

Every HTML page has this structure:
<!DOCTYPE html>
<html>
  <head> → Page info
  <body> → Page content
</html>

Basic HTML Tags

Common tags include:
• <h1> to <h6> → Headings
• <p> → Paragraph
• <br> → Line break
• <hr> → Horizontal line
• <strong> → Bold text
• <em> → Italic text

HTML Attributes

Attributes give extra information to tags.
Example:
<img src="image.jpg" width="200" alt="photo">

Lists in HTML

Ordered List (numbers):
<ol><li>Item</li></ol>

Unordered List (bullets):
<ul><li>Item</li></ul>

Links and Images

Link:
<a href="https://google.com">Go to Google</a>

Image:
<img src="photo.jpg">

Tables in HTML

Tables organize data:
<table>
  <tr><th>Name</th></tr>
  <tr><td>Ali</td></tr>
</table>

Forms in HTML

Forms collect user input:
<form>
  <input type="text">
  <input type="submit">
</form>

Semantic HTML Tags

Semantic tags give meaning:
• <header>
• <nav>
• <section>
• <article>
• <footer>

HTML Comments

Comments are not shown on the page:
<!-- This is a comment -->

HTML vs CSS vs JavaScript

• HTML → Structure
• CSS → Design
• JavaScript → Interaction

Conclusion

HTML is the foundation of web development.
Learning HTML is the first step to becoming a web developer.
